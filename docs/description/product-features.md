# Product Features

This section highlights the new features introduced in Deka LLM:

- **Enhanced Monitoring Dashboards:**  
  The service now provides a comprehensive Activity Monitoring Dashboard for customers, allowing them to track and analyze activities across all models in real time, specifically including coin and token usage metrics.
  - **Token-Based Usage Monitoring:** Users can view metrics such as Total Token Spent, Input & Output Token Usage, Top Models, Top API keys, and Monthly Token Spent.
  - **Coin-Based Usage Monitoring:** Users can view metrics such as Total Coin Spent, Top Models, Top API keys, and Monthly Coin Spent.

- **Alerting System for Coin Balance:**  
  A new alerting system provides warnings to customers when their coin balance reaches certain predefined thresholds. Users can configure these coin balance alerting rules. Both Useradmin and default users will receive email notifications when the coin balance reaches a configured threshold.

- **Coin Voucher Management (Superadmin):**  
  Superadmins now could create vouchers for DekaLLM usage, which users can redeem to receive coin balance.
