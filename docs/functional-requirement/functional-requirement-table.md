# Functional Requirement Table

| ID         | Functional Requirement            | Description |
|------------|----------------------------------|-------------|
| LLM-UA-011 | Coin Monitoring                  | Useradmin can view the metrics of usage activity API requests based on coin balance. Metrics including Total Coin Spent, Top Models, Top API keys, Monthly Coin Spent. You can see in the Appendix in sub-chapter 6.2 Usage Dashboard Mockup – Coin Filter. |
| LLM-UA-012 | Token Monitoring                 | Useradmin can view the metrics of usage activity API requests based on token. Metrics include Total Token Spent, Input & Output Token Usage, Top Models, Top API keys, Monthly Token Spent. You can see in the Appendix in sub-chapter 6.1 Usage Dashboard Mockup – Token Filter. |
| LLM-UA-013 | Alerting Configure               | Useradmin can configure coin balance alerting rules. You can see in the Appendix in Figure 3. Alerting Mockup. |
| LLM-UA-014 | Alerting Notification            | Useradmin and default users will receive an email notification when their coin balance reaches a threshold defined by the coin balance alerting rule. Email should be sent within 5 minutes of reaching the defined thresholds. |
| LLM-SA-011 | Superadmin Coin Monitoring       | Superadmin can view the metrics of usage activity API requests based on coin balance. Metrics including Total Coin Spent, Top Models, Top API keys, Monthly Coin Spent. You can see in the Appendix in sub-chapter 6.2 Usage Dashboard Mockup – Coin Filter. |
| LLM-SA-012 | Superadmin Token Monitoring      | Superadmin can view the metrics of usage activity API requests based on token. Metrics include Total Token Spent, Input & Output Token Usage, Top Models, Top API keys, Monthly Token Spent. You can see in the Appendix in sub-chapter 6.1 Usage Dashboard Mockup – Token Filter. |
| LLM-SA-013 | Alerting Configuration           | Superadmin can configure coin balance alerting rules. You can see in the Appendix in sub-chapter 6.3 Alerting Mockup |
| LLM-SA-014 | Vouchers                         | Superadmin can create vouchers for DekaLLM usage, users redeeming code will receive coin balance |
